## Getting Started with Notebooks

1. Install Python 3.10+ and Jupyter:
   ```bash
   pip install jupyter numpy scipy matplotlib pandas
   ```

2. Clone this repository:
   ```bash
   git clone https://github.com/ExaPsi/2302337.git
   cd 2302337/notebooks
   jupyter notebook
   ```

3. Or open in Google Colab — click any `.ipynb` file on GitHub, then click the "Open in Colab" badge.
